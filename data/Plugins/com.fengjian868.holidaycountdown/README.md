# 节假日倒计时插件

## 功能模块

- **节假日倒计时** - 显示距离最近节假日的倒计时，支持弧形进度环、调休提醒、放假天数
- **24节气倒计时** - 显示当前节气及下一个节气倒计时，支持自定义颜色
- **农历日期显示** - 支持多种显示格式（完整/简洁/含生肖/含节气），可自定义模板
- **自定义节日倒计时** - 添加你自己的节日（生日、纪念日等），支持每年重复
- **寒暑假倒计时** - 显示距离暑假/寒假的剩余时间（周+天）
- **时段问候语** - 根据时间显示早中晚问候、放学提醒、周日晚修提示
- **天气问候** - 根据 ClassIsland 天气温度自动显示穿衣提醒

## 使用说明

### 节假日倒计时组件
- 自动显示最近的节假日及倒计时
- 首个节日可显示弧形进度环（圈内显示日期）
- 支持调休上课提醒
- 显示该节日放假天数

### 自定义节日
- 在设置页添加你的节日（支持每年重复）
- 组件会横向排列显示多个节日

### 问候语设置
- **时段文案**：自定义多个时间段的问候语（如早/中/晚）
- **特殊日期**：设置特定星期几的问候语，可指定时段（如"周五下午"）
- 特殊日期优先级高于时段文案
- 开启"联网刷新"后每天自动获取新文案

### 农历设置
- 选择预设格式或自定义模板
- 可用变量：`{gzYear}` 干支年、`{IMonthCn}` 农历月、`{IDayCn}` 农历日、`{Animal}` 生肖、`{Term}` 节气
